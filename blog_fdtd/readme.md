# Read me

fdtd_notebook.ipynb is an acoustic simulation program powered by FDTD method.

This program is derived from "sampleFDTD_rectangle.py" made by [openacoustics project](<http://www.openacoustics.org/>) .

## need package

- numpy
- scipy
- matplotlib
